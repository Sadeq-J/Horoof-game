#Horoof game

## Technologies Used

- HTML
- CSS
- JavaScript

## Description
Horoof is a two-team Arabic word game inspired by connection-based strategy games. Teams compete to connect opposite sides of the board by correctly answering questions based on selected Arabic letters.

## User Stories
- As a user, I want to click on a letter cell to display its question.
- As a user, I want to reveal the correct answer.
- As a user, I want to assign the letter to the team who brought the correct answer.
- As a user, I want to see which team’s turn it is to choose the letter.
- As a user, I want to know when a team has won the game.
